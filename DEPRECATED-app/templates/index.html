"Analyze, Perfect and Visualize Your User Stories with GRIMM"

Quality User Story Framework

A User Story Framework

Quality User Story Framework is a collection of quality criteria for individual user stories as well as sets of user stories. Prevent ambiguity from turning into wrong assumptions by teaching this simple framework to the entire software development team.

Read More

AQUSA

Automated Syntacical User Story Analysis

Our first tool, AQUSA, helps you apply the QUS-framework by analyzing your user stories and providing feedback on how to improve them. It highlights your areas of improvement. With AQUSA, you will really make the perfect user stories.

Start Using AQUSA

Visual Narrator

Conceptual Model Generation

At the centre of the GRIMM toolkit is Visual Narrator, which uses Natural Language processing techniques to analyze your user stories. The tool can generate functional models in the form of HTML-reports and Ontologies.

GitHub

Interactive Narrator

Interactive Conceptual Models

Interactive Narrator represents the final stage of the GRIMM-process. It visualizes your user stories into a touch screen friendly interactive diagram. Interactive Narrator helps initiate fruitful discussion about software requirements.

About Interactive Narrator

Interactive Narrator is a documentation tool that makes your user stories visual, facilitating discussion about requirements.

Interactive Narrator offers users the ability to upload their user stories in .csv or .txt format and will automatically call on a Natural Language Processing tool called Visual Narrator to derive a conceptual model and generate a graphical representation of that model. This model is then presented to the user in the shape of a network diagram with which users can interact.

The tool provides filter mechanisms that allow hiding or showing sprints and roles within and between sets of stories, a search function to find concepts or relationships, and the option to turn relationships on or off. Users can also zoom in and out to inspect certain parts of the model. The visualization itself is a network diagram that consists of nodes that represent important concepts from the user stories and edges that represent the relationships between those concepts. When clicking on a concept (node)in the diagram, the full user stories in which the concept occurs are shown, providing the user with more details.

Interactive Narrator has been presented at REFSQ 2018 conference in Utrecht, Netherlands. See the tool paper here.
